            # 🏋️‍♂️ Gym Manager – تطبيق إدارة الجيم

تطبيق سطح مكتب احترافي لإدارة بيانات مشتركين الجيم باستخدام **Python**, **PyQt6** و **SQLite**.  
يساعدك على تنظيم البيانات، إضافة مشتركين جدد، البحث، التعديل، والعرض بطريقة سهلة وسريعة.
